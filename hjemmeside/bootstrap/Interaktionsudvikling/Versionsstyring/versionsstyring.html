Versionsstyring




Tilbage til forsiden

Versionsstyring benyttes til at danne overblik over og have styr på forskellige versioner af det samme dokument eller flere delprojekter. Tilgangen er metodisk hvilket vil sige at overblikket kan dannes ved hjælp af noget så simpelt som navngivning af filer og mapper. Versionsstyring bruges meget i forbindelse med kodning og programmering, og er yderst relevant samt hjælpsomt, da det muliggør at gå tilbage til en tidligere version af et dokument, skulle man fortryde de ændringer der er foretaget. Det bruges også i forbindelse med samarbejde; er man flere personer om de samme dokumenter, kan man benytte sig af forskellige webplatforme, som f.eks. GitHub, til at dele og ændre dokumenterne.


Git

Git er et versionsstyingssystem som sætter standarden for online tjenester som GitHub, GitLap osv.


GitHub

GitHub er et socialt medie for programmører og folk der gerne vil dele deres kode med andre. Her er det muligt at oprette Git-repositories, en server i skyen, så det ikke er nødvendigt at installere en. Uploader man en fil på GitHub er de som udgangspunkt tilgængelige for alle, og mange bruger derfor GitHub som et slags curriculum vitae og/eller potfolio hvor det er muligt at vise hvordan ens personlige kompetencer ligger.
